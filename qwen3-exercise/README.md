# Running Qwen3.5 Locally (CPU)

## Objective

The objective of this exercise is to run an open-source Large Language Model locally without using cloud APIs and evaluate its performance on different prompt categories.

The model used in this experiment is:

Qwen3.5-0.8B

This is a lightweight model designed to run efficiently on consumer hardware.

---

# System Specifications

CPU: Intel Core i5  
RAM: 16 GB  
GPU: None (CPU inference only)

Operating System: Windows 11

---

# Framework Used

Hugging Face Transformers

Other libraries used:

- PyTorch
- Accelerate
- Transformers

---

# How the Model Was Run

The model was executed locally using Python and Hugging Face Transformers.

Steps followed:

1. Installed dependencies
2. Downloaded the model from HuggingFace
3. Loaded the tokenizer and model
4. Ran inference using CPU
